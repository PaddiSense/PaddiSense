# PaddiSense Farm Management

Modular farm management platform for Home Assistant.

## Features

- **Farm Registry** - Manage paddocks, bays, and farm infrastructure
- **Weather Module** - Local weather monitoring and forecasting
- **HFM (Hay & Fodder)** - Fertilizer and hay cutting management
- **IPM (Irrigation)** - Irrigation scheduling and monitoring
- **PWM (Pump/Valve Control)** - ESPHome device control

## Installation

1. Add this repository to HACS
2. Download PaddiSense
3. Restart Home Assistant
4. Go to Settings > Devices & Services > Add Integration > PaddiSense
5. Follow the setup wizard

## Requirements

- Home Assistant 2024.1.0 or newer
- Git (for module downloads)
